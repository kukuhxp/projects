# BOOLEAN

## Boolean in Number

Example:

```
false == 0; // true
false === 1; // false
true == 1; // true
true === 1; // false

/**
 * Operator == melakukan type coercion, yaitu mengubah tipe data supaya bisa dibandingkan.
 * True otomatis dikonversi menjadi 1 saat dibandingkan dengan angka. Jadi true == 1 adalah benar.
 */
```
