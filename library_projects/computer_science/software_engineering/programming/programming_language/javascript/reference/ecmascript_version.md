# ECMASCRIPT VERSION

## 1. ECMAScript 5 (ES5 / 2009)

- New directive:
   
   ```
   - "use strict"
   ```
   
- Index string.
- Template literal.
- New string method:
  
  ```
   - String.prototype.trim()
  ```
  
- New array instance methods:

  ```
   - Array.prototype.forEach()
   - Array.prototype.map()
   - Array.prototype.filter()
   - Array.prototype.reduce()
   - Array.prototype.every()
   ```

- New array static method:

  ```
  - isArray()
  ```
   
- New JSON methods:

   ```
   - JSON.parse()
   - JSON.stringify()
   ```
- New object methods:

   ```
   - Object.create()
   - Object.keys()
   - Object.defineProperty()
   ```
   
- New function instance method:

   ```
   - Function.prototype.bind()
   ```
   
- Trailing commas.
