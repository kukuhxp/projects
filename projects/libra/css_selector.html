Cascading Style Sheet Selector Table

Jenis Selector Contoh Fungsi
Universal Selector * Memilih semua elemen
Type Selector p, h1, div Memilih elemen berdasarkan nama tag
Class Selector .judul Memilih elemen dengan class tertentu
ID Selector #header Memilih elemen berdasarkan ID
Descendant Selector div p Memilih p di dalam div
Child Selector ul > li Memilih li yang merupakan anak langsung dari ul
Adjacent Sibling h1 + p Memilih p yang langsung setelah h1
General Sibling h1 ~ p Memilih semua p yang bersaudara dengan h1
Attribute Selector [type="text"] Memilih elemen input dengan tipe teks
[href^="https"] Memilih elemen dengan atribut href diawali "https"
[href$=".pdf"] Memilih elemen dengan atribut href diakhiri ".pdf"
[title*="tips"] Memilih elemen dengan atribut title mengandung "tips"
Pseudo-class: State a:hover, input:focus Memilih elemen dalam kondisi tertentu (hover, focus, dll)
Pseudo-class: Posisi li:first-child, li:last-child Memilih elemen berdasarkan posisi
li:nth-child(2) Memilih elemen anak ke-2
li:nth-of-type(odd) Memilih elemen berjenis sama di posisi ganjil
:not(selector) :not(.aktif) Memilih elemen yang bukan class .aktif
:is(), :where() :is(h1, h2) Memilih elemen yang cocok dari daftar selector
Pseudo-element ::before, ::after Menyisipkan konten sebelum/sesudah elemen
::first-line Mengatur baris pertama dari teks
::placeholder Mengatur style teks placeholder input
@media Query @media (max-width: 768px) Menentukan style khusus untuk ukuran layar tertentu (responsif)
@media screen and (orientation: portrait) Menerapkan style saat layar dalam mode potret
