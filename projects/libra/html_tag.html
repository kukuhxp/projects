HTML Tag Table

x
    Category Tag Descriotion
    Root <html> Elemen akar dari dokumen HTML
    Metadata <head> Kontainer untuk metadata dokumen
    <title> Judul dokumen yang ditampilkan di tab browser
    <base> Menentukan URL dasar untuk semua URL relatif dalam dokumen
    <link> Menentukan hubungan antara dokumen saat ini dan sumber daya eksternal
    <meta> Menyediakan metadata seperti deskripsi, kata kunci, dan penulis dokumen
    Struktur <body> Menentukan isi utama dokumen HTML
    <header> Menentukan bagian header untuk dokumen atau bagian
    <nav> Menentukan navigasi utama dalam dokumen
    <section> Menentukan bagian dalam dokumen
    <article> Menentukan konten independen seperti artikel
    <footer> Menentukan bagian footer untuk dokumen atau bagian
    Konten Teks <h1> - <h6> Menentukan heading tingkat 1 hingga 6
    <p> Menentukan paragraf teks
    <blockquote> Menentukan kutipan blok dari sumber lain
    <ol> Menentukan daftar bernomor (ordered list)
    <ul> Menentukan daftar berpoin (unordered list)
    Semantik Teks Inline <a> Menentukan hyperlink
    <strong> Menentukan teks dengan penekanan kuat (biasanya tebal)
    <em> Menentukan teks dengan penekanan (biasanya miring)
    <mark> Menyoroti teks
    <span> Kontainer umum untuk mengelompokkan teks untuk tujuan styling
    Gambar dan Multimedia <img> Menyisipkan gambar
    <audio> Menyisipkan konten audio
    <video> Menyisipkan konten video
    Formulir <form> Menentukan formulir untuk input pengguna
    <input> Menentukan kontrol input (teks, radio, checkbox, dll)
    <textarea> Area input teks multiline
    <button> Tombol untuk submit atau aksi lainnya
    <select> & <option> Dropdown pilihan dan item pilihannya
    Tabel <table> Membuat tabel
    <tr> Baris dalam tabel
    <td> Sel data dalam tabel
    <th> Sel header dalam tabel
    <thead>, <tbody>, <tfoot> Bagian kepala, isi, dan kaki tabel
    Skrip <script> Menyisipkan atau merujuk ke JavaScript
    <noscript> Konten alternatif saat JavaScript tidak didukung
    Lainnya <iframe> Menyisipkan dokumen HTML lain
    <canvas> Area gambar dinamis melalui JavaScript
    <details> & <summary> Elemen interaktif untuk membuka/tutup konten
    <template> Menentukan fragment HTML untuk digunakan nanti (JavaScript)
